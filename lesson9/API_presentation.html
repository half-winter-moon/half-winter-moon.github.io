APIs

Presentation Video

There are four main kinds of APIs:

  1. Open APIs
  2. Partner APIs
  3. Internal APIs
  4. Composite APIs

An API is an Application Programming Interface.

An API can do many things. You can interact with them using JavaScript. Usually what you do is send out a message or "request" to the API, and then the response is what your code will work with. The fetch() method is a common way to reach out to an API, with at least one argument for the path to the resource being required.

The result of such a fetch() method can be JSON, or JavaScript Object Notation, which is text that contains data in a specific format.
